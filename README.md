#  Online Car Booking & Rental Management System

##  Overview

The **Online Car Booking & Rental Management System** is a web-based platform designed to simplify the process of renting cars online. The system allows users to browse available vehicles, book cars based on availability, and manage bookings through a user-friendly interface.

The platform is designed to reduce manual effort, save time, and provide a seamless booking experience for customers while allowing administrators to efficiently manage vehicles, bookings, and users.

---

##  Objectives

- To provide an easy-to-use online car booking platform.
- To allow users to search and book cars from anywhere.
- To enable administrators to manage vehicles, bookings, and users efficiently.
- To reduce paperwork and manual rental management processes.
- To improve customer satisfaction through a digital solution.

---

##  User Roles

The system supports three types of users:

###  Admin
- Manage car brands and car information
- Add, update, or delete cars
- Manage booking requests
- Confirm or cancel bookings
- Manage payment records
- Manage user information
- View and respond to customer feedback

###  Registered User
- Register and log in
- Search available cars
- View car details
- Book cars
- Make payment
- Update profile
- Change password
- Submit feedback

###  Guest
- Browse the website
- View car listings
- Must register to book a car

---

##  System Workflow

1. User registers or logs into the system.
2. User searches for available cars.
3. User selects a car.
4. User books the car by providing required details.
5. Admin reviews booking request.
6. Admin confirms or cancels booking.
7. Payment record is generated.
8. Booking details are stored in the database.
